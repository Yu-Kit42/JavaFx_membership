# JavaFx_membership
### JavaFx & 오라클DB를 활용, 씬빌더로 구현
회원가입 버튼을 통하여 DB에 데이터를 업로드
로그인 시 DB데이터와 일치하면 입력한 데이터 출력 


# 사용 언어
 javaFx + SceneBuiler + Oracle
